# REOR

A web app for ordering dishes in a restaurant

## Prerequisites

- To use AWS SES:
    - Set env var:
        - ```AWS_ACCESS_KEY_ID```
        - ```AWS_SECRET_ACCESS_KEY```

- To use Telegram bot:
    - Create Telegram bot
    - Invite bot into a group
    - Set bot as administrator in the group
    - Set env var: ```TELEGRAM_BOT_TOKEN```
